# Wazuh Dashboard

## Overview

The Wazuh Dashboard provides a centralized web interface for monitoring, analyzing, and investigating security events collected from the Host, Network, and Cloud layers. It allows security analysts to view security alerts, monitor connected agents, search logs, investigate incidents, and analyze system activities in real time.

In the proposed system, the dashboard acts as the primary interface for visualizing both rule-based alerts generated by the Wazuh Manager and AI-generated alerts produced by the Machine Learning Engine. This enables centralized monitoring of all monitored environments through a single platform.

---

# Dashboard Features

The Wazuh Dashboard provides several features that assist security analysts in monitoring and investigating security events.

## Real-Time Security Monitoring

The dashboard continuously receives processed security events from the Wazuh Manager, allowing analysts to monitor the security status of all connected endpoints in real time.

---

## Agent Monitoring

The dashboard displays all registered Wazuh Agents along with their connection status, operating system, IP address, and synchronization status.

This allows administrators to verify that all monitored systems are successfully connected to the centralized Wazuh Server.

---

## Log Management

Security logs collected from the Host, Network, and Cloud layers are indexed and displayed within the dashboard.

Analysts can:

- Search logs
- Filter events
- View event details
- Investigate suspicious activities
- Analyze historical events

---

## Security Alerts

Whenever suspicious activity is detected, Wazuh generates security alerts based on predefined detection rules.

The dashboard displays:

- Alert severity
- Rule ID
- Rule description
- Timestamp
- Agent information
- Event details

These alerts help analysts quickly identify potential security incidents.

---

## AI-Based Alerts

In addition to traditional Wazuh alerts, the dashboard also displays alerts generated by the AI Engine.

After analyzing security logs using machine learning models, the AI Engine sends anomaly detection results back to the Wazuh Manager. These alerts are then stored and visualized alongside rule-based alerts, providing enhanced threat detection capabilities.

# Summary

The Wazuh Dashboard serves as the central monitoring interface of the proposed SIEM architecture. It consolidates security events collected from multiple environments, enabling analysts to monitor infrastructure, investigate incidents, and visualize both rule-based and AI-generated alerts in real time. By integrating traditional SIEM capabilities with machine learning-based anomaly detection, the dashboard provides a comprehensive platform for proactive threat detection and security monitoring.
