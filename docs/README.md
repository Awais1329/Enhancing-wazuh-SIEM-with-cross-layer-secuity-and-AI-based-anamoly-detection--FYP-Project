# Project Documentation

This folder contains the technical documentation for the **Enhancing Wazuh SIEM with Cross-Layer Security and AI-Based Anomaly Detection** project.

The documentation explains the complete deployment process, system configuration, AI integration, dashboard monitoring, and troubleshooting steps required to reproduce the proposed system.

## Documentation Contents

| Document | Description |
|----------|-------------|
| Installation.md | Installation of Ubuntu, Wazuh, Suricata, and project dependencies. |
| Wazuh-Deployment.md | Deployment and configuration of the Wazuh Manager, Indexer, and Dashboard. |
| Agent-Configuration.md | Installation and configuration of Wazuh Agents for the Host, Network, and Cloud layers. |
| AI-Engine.md | AI pipeline, preprocessing, machine learning models, and alert generation. |
| Dashboard.md | Overview of the Wazuh Dashboard and security monitoring features. |
| Attack-Simulation.md | Attack scenarios used to evaluate the system and generate security events. |
| Troubleshooting.md | Common installation, configuration, and deployment issues with their solutions. |

## Purpose

The documentation is intended to help researchers, students, and cybersecurity practitioners understand the implementation of the proposed SIEM architecture and reproduce the complete system in their own environment.
