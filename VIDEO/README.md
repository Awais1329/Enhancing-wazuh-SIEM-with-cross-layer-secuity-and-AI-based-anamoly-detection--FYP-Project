# Project Demonstration

A demonstration video of the proposed **Enhancing Wazuh SIEM with Cross-Layer Security and AI-Based Anomaly Detection** system is included in this repository.

The video provides a complete walkthrough of the project implementation, showcasing the deployment and configuration of the Wazuh Server, registration of Wazuh Agents across the **Host**, **Network**, and **Cloud (OpenStack)** layers, and the successful collection of security logs from all monitored environments. It also demonstrates the implementation of the AI Engine, including the machine learning code, log preprocessing, anomaly detection process, and the generation of AI-based security alerts. Finally, the video illustrates how both rule-based and AI-generated alerts are integrated into the Wazuh SIEM and visualized on the Wazuh Dashboard for real-time security monitoring and analysis.

## Demo Highlights

- Wazuh Server deployment and configuration
- Host, Network, and Cloud agent registration
- Multi-layer security log collection
- Wazuh Manager log processing
- AI Engine implementation and source code
- Machine learning-based anomaly detection
- AI-generated alert integration with Wazuh
- Real-time visualization on the Wazuh Dashboard
